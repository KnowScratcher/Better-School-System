# 必要事項
- 必須安裝tampermonkey
- Chrome使用者必須在[插件管理員](chrome://extensions/)開啟**開發人員模式**

# 安裝
請見[安裝](https://github.com/KnowScratcher/Better-School-System/wiki/%E5%AE%89%E8%A3%9D)


# 原理
幫你加一些功能上去，像是保持登入，不會20分鐘就登出。

**簡單來說就是外掛**
